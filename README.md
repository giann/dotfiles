## How to use

Use [GNU Stow](https://www.gnu.org/software/stow/) to apply what you need:

```bash
stow awesome
```

## Included

- Awesome WM
- Compton
- DCSS
- Fonts
- gnome, gtk & X themes
- ncmpcpp
- neovim
- ranger
- sublime text
- termite
- tig
- tmux
- zsh + oh-my-zsh
